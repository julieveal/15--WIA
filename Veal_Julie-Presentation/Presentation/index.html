Jeditable

A jquery plugin

by Julie Veal

What is Jeditable?

Jeditable is inplace editor for jQuery. You can click and edit html elements right on the web page. When the user clicks the block of text it becomes a form. User then presses submit. The info is sent to the webserver and saved.

Why Jeditable?

I chose Jeditable because it is an inplace editor plugin for jQuery. I can see myself using it to do children's ebooks where a child can input their name, etc. and make their own ebook online for print. It can also be used anywhere there is a need for an online form.

Pros and Cons

    Pros -
  1. You can get full control of the Ajax request by submitting to the function instead of URL. Parameters passed are same as with callback.
  2. There are several custom inputs: inlined select, normal, double-click and mouse-over.

    Cons -
  1. Some people are concerned about extra request made to server. You can also combine these two approaches. Let PHP output JSON encoded array directly into JavaScript code.

$(document).ready(function() {
	$('.edit').editable('http://www.example.com/save.php', {
	indicator : 'Saving...',
	tooltip   : 'Click to edit...'
	});
$('.edit_area').editable('http://www.example.com/save.php', {
	type      : 'textarea',
	cancel    : 'Cancel',
	submit    : 'OK',
	indicator : '',
	tooltip   : 'Click to edit...'
	});
});
